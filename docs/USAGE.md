# Neon Equalizer Usage Guide

## First Run

1. Install Equalizer APO.
2. Open Neon Equalizer.
3. Confirm the detected config folder, or click the config path control to choose the Equalizer APO `config` directory.
4. Start with **Load Preset...** or create your own filters in the **Equalizer** tab.

## Editing EQ

- Use **Parametric EQ** for precise filters such as peaking, low-shelf, high-shelf, low-pass, and high-pass.
- Use **Graphic EQ** for broader tone shaping.
- Drag graph handles to adjust filter frequency and gain visually.
- Toggle filters with the switch on each row.
- Use **Auto** preamp to reduce clipping risk.

## AutoEQ And Targets

1. Open **AutoEQ & Headphones**.
2. Choose a measurement source or target curve.
3. Adjust target preferences if needed.
4. Generate filters and review the graph before saving.

## Previewing Audio

1. Open the **EQ Preview** player at the bottom of the app.
2. Choose pink noise, white noise, or load an audio file.
3. Toggle EQ preview on and off to hear the effect before writing to Equalizer APO.

## Saving To Equalizer APO

1. Choose the target device from the top bar.
2. Click **Save to APO**.
3. Restart audio playback if your player does not pick up the config change immediately.

## Presets

- Save reusable EQ setups with **Save EQ**.
- Load saved presets from the app or tray menu.
- When upgrading from the old Equalizer APO Studio identity, Neon Equalizer copies old presets forward if the new preset folder is empty.

## Troubleshooting

- If the app cannot save, run it as administrator.
- If no system-wide EQ is heard, open Equalizer APO Configurator and make sure your playback device is selected.
- If the graph looks clipped, lower the preamp or enable automatic preamp.
- If a target or measurement does not load, check your internet connection and try another source.
